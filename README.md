# Meus Mods de Jogos

Este repositório centraliza as modificações que desenvolvi para diversos jogos. O foco é reunir melhorias, novas funcionalidades e correções, independente do motor ou gênero do jogo.

---

## Estrutura do Repositório

Para manter a organização, o repositório segue este padrão:

*   **Pastas por Jogo:** Cada jogo possui sua própria pasta dedicada.
*   **Instalações Individuais:** Dentro de cada pasta, você encontrará um arquivo `README.md` específico com o tutorial de instalação e detalhes daquele mod.
*   **Links Diretos:** Não é necessário procurar arquivos na aba de Releases; cada README de jogo já contém o **link direto para o download** da versão mais atualizada.
*   **Código Fonte:** Sempre que possível e viável, os scripts ou códigos-fonte das modificações são disponibilizados nas pastas para transparência e estudo.
