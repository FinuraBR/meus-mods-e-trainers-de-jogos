# My Game Mods and Trainers

**[Versão em Português](./README.md)**

This repository centralizes the modifications and trainers I've developed for various games. The focus is on gathering enhancements, new functionalities, fixes, and utilities (such as cheats/trainers), regardless of the game engine or genre.

---

## Repository Structure

To maintain organization, the repository follows this pattern:

*   **Game-Specific Folders:** Each game has its own dedicated folder.
*   **Individual Installations:** Inside each folder, you will find a specific `README.md` file with installation/usage tutorials and details for that particular mod or trainer.
*   **Direct Links:** There's no need to search for files in the Releases tab; each game's README already contains a **direct download link** to the most updated version.
*   **Source Code:** Whenever possible and viable, the scripts or source code for the modifications and trainers are provided in the folders for transparency and study.
